# Capítulo 1

## Tema

Introdução aos códigos e à comunicação de informações.

## Objetivos

- Entender o conceito de código.
- Compreender como mensagens podem ser representadas.
- Iniciar o raciocínio binário.

## Conceitos estudados

- Código
- Comunicação
- Representação
- Informação

## Aplicação em Segurança

Toda comunicação em computadores é baseada em representação de informações.

Entender isso será essencial para:

- Redes
- Criptografia
- Engenharia Reversa
- Exploração de protocolos
