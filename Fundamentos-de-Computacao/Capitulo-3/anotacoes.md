O que aprendi

O código Braille é também um código binário.
Através de 64 códigos diferentes, consegue reproduzir as letras do alfabeto latino, números, caracteres especiais, etc...
Acredito que a principal sacada do Braille seja usar letras para representar as próprias letras que podem virar palavras, mas que as próprias letras podem
representar uma palavra inteira. Não se aplica a todas as letras, mas, por exemplo, a letra b pode ser entendida como "but", a letra c como "can", a letra d como
"do", etc...
Existem códigos que ao serem aplicados antes de uma letra ou número, podem representar uma troca no código a seguir, trocando por exemplo, uma letra minúscula por
maiúscula. Isso ocorre quando aplicado o código "casa 6 elevada" (foi usado o termo casa, pois são 6 posições e fica melhor o entendimento) ou "dot 6 raised" na
frente de outro código de letra.
